로봇대회 기록 경기 순위표

종목명, 심사위원명, 팀명, 1차·2차 기록을 입력하면 기록 유형과 순위 산정 기준에 맞춰 순위를 자동으로 계산합니다. 시간 기록과 점수 기록을 모두 지원하며 결과 화면은 PDF 또는 PNG로 저장할 수 있습니다.

대회 설정

순위 계산 기준을 먼저 정한 뒤 아래 참가팀 기록을 입력하면 오른쪽 순위표가 즉시 갱신됩니다.

결과표 제목과 출력 파일 이름에 사용됩니다.
쉼표 또는 줄바꿈으로 여러 명을 입력할 수 있습니다.
시간 기록 정렬 안내 시간 기록은 숫자가 낮을수록 상위 순위입니다. 최고 기록은 더 빠른 기록을 뜻합니다.

참가팀 기록 입력

행을 추가해서 팀별 1차, 2차 기록을 넣어주세요. 빈 기록은 미입력으로 처리됩니다.

각 팀마다 1차 점수, 1차 시간, 2차 점수, 2차 시간을 모두 입력할 수 있습니다. 점수는 숫자만, 시간은 `17.92` 또는 `1:05.33`처럼 입력하세요.
번호 팀명 1차 점수 1차 시간 2차 점수 2차 시간 삭제

자동 순위 결과

입력한 두 차례 기록을 기준으로 순위가 자동 계산됩니다. 출력할 때는 아래 저장 버튼을 사용하세요.

순위 산정 설정

출력물에도 현재 설정이 함께 표시됩니다.

최종 순위표

기록을 입력하면 순위표가 생성됩니다.
