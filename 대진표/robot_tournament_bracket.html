로봇경진대회 대진표 UI

참가 팀 수를 고르고, 진행 방식과 팀명을 입력한 뒤 랜덤 대진표를 생성하세요. 팀 수에 따라 그룹 스테이지, 녹아웃 스테이지, 그룹 + 녹아웃 혼합 방식이 자동으로 추천되며 토너먼트에서는 부전승도 반영됩니다.

대회 설정

팀 수를 정하면 입력 칸과 선택 가능한 경기 방식이 함께 바뀝니다.

출력용 결과지 제목과 최종 결과표에 표시됩니다.
여러 명이면 쉼표 또는 줄바꿈으로 입력하세요.

3. 참가팀명 입력

빈칸은 자동으로 `팀 1`, `팀 2` 형식으로 채워집니다.

생성 결과

그룹전 일정표와 녹아웃 대진은 선택한 방식에 맞춰 바로 렌더링됩니다.

참가 팀 - 팀 수를 선택하면 여기에 요약이 표시됩니다.
진행 방식 - 가능한 방식은 팀 수에 따라 달라집니다.
대진 상태 대기 중 랜덤 대진표 생성 버튼을 누르면 결과가 나타납니다.
왼쪽에서 참가 팀 수와 경기 방식을 고른 뒤 팀명을 입력하고 `랜덤 대진표 생성`을 눌러주세요.
